## Requirements for development
1. Visual Studio Code
2. Docker
3. NodeJS (8 or later)


## Start Development environment
1. Open `CMD` or `Terminal`
2. Browser to code repository (`ex. C:\git\mobacon`)
3. Type `docker-compose up`
